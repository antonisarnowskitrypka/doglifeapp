# Demo Page

A public page showing a prospective provider how the platform works — on fictional data and with simulated events, without signing up.

## What it is

- A **guided tour** of a ready-made demo organization with realistic, fully populated data (services, calendar, clients, pets, history).
- The data is **read-only** — the visitor browses real provider screens, but nothing they do is saved.

## Event simulation

The visitor can trigger events with buttons and watch the panel react in real time, for example:

- "Simulate a new booking" → a booking appears, the calendar updates, a notification fires
- "Simulate a payment" → escrow/earnings update
- "Simulate a review" → a new review appears on the profile

This shows the live flow (booking → payment → session → review) on demand.

## Safety

- All data is fictional and **isolated** — it never touches real production data.
- Each visit is **ephemeral**: simulated changes live only within that session and are discarded afterwards.
- No real payments, emails or push notifications are sent.

## Purpose

To let a prospective provider experience the panel and the full flow before deciding to create an account.
