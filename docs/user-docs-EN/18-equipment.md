# Equipment Registry

A tool for providers to keep an inventory of their equipment, service history, and inspection reminders. It's especially handy for physiotherapists (underwater treadmills, Indiba) and sports trainers (agility gear).

## Adding equipment

A provider can add equipment items to their organization. Each item can include:

- Name and description
- Purchase / acquisition details
- Service history
- Inspection / maintenance reminders

## Public visibility

Each equipment item has a visibility toggle (the "eye"). When turned on, the item is shown publicly on the provider's profile — useful for advertising your capabilities (e.g. "we have an underwater treadmill").

When hidden, the item is used purely for internal record-keeping.

Public equipment also **boosts search** — its names feed into the company's tags, so clients searching for, say, "underwater treadmill" can find you more easily. Hidden equipment doesn't affect search.

## Service and maintenance

For each item, the provider can log service events and set reminders for upcoming inspections or maintenance.

- Reminders are delivered **in-app**.
- Every record (the item itself and each service event) supports **file attachments** — invoices, service reports, inspection certificates, photos.

## Reminders

When an inspection/maintenance date is approaching, the provider receives an in-app notification (see [Notifications](./16-notifications.md)).
