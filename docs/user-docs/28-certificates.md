# Certyfikaty

Za **eventy** i **kursy (pakiety)** platforma generuje certyfikaty PDF.

## Kiedy powstają

Automatycznie po ukończeniu:

- **Event** — po jego zakończeniu, dla zapisanych uczestników.
- **Kurs (pakiet)** — po wykorzystaniu wszystkich sesji.

Klient dostaje powiadomienie, a certyfikat (PDF) trafia do jego dokumentów.

## Co jest na certyfikacie

Certyfikat powstaje na **z góry ustalonym przez platformę tle**, z predefiniowanymi miejscami na:

- **właściciela i zwierzaka** (np. „Jan Kowalski z psem Lanza"),
- nazwę kursu / eventu,
- nazwę szkoły (organizacji),
- logo,
- datę,
- **podpis** usługodawcy.

W ustawieniach usługodawca wgrywa swój **podpis w PNG** oraz logo.

## Po MVP

Własne, customowe szablony certyfikatów.
