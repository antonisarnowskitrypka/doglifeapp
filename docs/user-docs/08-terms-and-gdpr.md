# Regulamin i RODO usługodawcy

Każdy usługodawca może opublikować własny **regulamin** oraz **politykę RODO / prywatności**, które klient akceptuje przy rezerwacji.

## Konfiguracja usługodawcy

Usługodawca może dodać i edytować:

- Swój **regulamin**
- Swoją **politykę RODO / prywatności**

## Akceptacja przez klienta

- Klient akceptuje regulamin i politykę usługodawcy podczas rezerwacji usługi.
- Akceptacja jest zapamiętana **per usługodawca** — klient nie jest pytany przy każdej rezerwacji.
- Jeśli usługodawca **opublikuje nową wersję**, klient zostanie poproszony o ponowną akceptację przy następnej rezerwacji.

## Zapis

Każda akceptacja jest zapisywana wraz z wersją dokumentu, dzięki czemu usługodawca ma dowód, na co i kiedy klient wyraził zgodę.
