# Przegląd platformy

Platforma typu marketplace + SaaS, która łączy właścicieli zwierząt z profesjonalnymi usługodawcami.

## Do czego służy

Właściciele zwierząt mogą znajdować, rezerwować i opłacać usługi dla swoich zwierzaków. Usługodawcy zarządzają w jednym miejscu kalendarzem, klientami, sesjami i płatnościami.

## Zakres MVP

1. Rezerwacja usług i marketplace
2. Płatności online
3. Kalendarze usługodawcy i pracowników
4. Przestrzeń sesji (notatki, prace domowe, załączniki)
5. Oś czasu historii zwierzaka
6. Wydarzenia grupowe
7. Pakiety treningowe

## Po MVP

- Abonament właściciela zwierzaka
- Rankingi podróży i aktywności zwierzaka
- Integracje weterynaryjne
- Funkcje zdrowotne w abonamencie
