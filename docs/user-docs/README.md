# Dokumentacja użytkownika

Przewodnik po platformie ułożony jak instrukcja — od podstaw, przez ścieżkę właściciela zwierzaka i usługodawcy, po pomoc i panel platformy.

> **Wersje językowe:** dokumentacja użytkownika jest dwujęzyczna — polska w `user-docs/` i angielska w [`user-docs-EN/`](../user-docs-EN/README.md) (te same nazwy plików). Każdy dokument ma na końcu nawigację **poprzedni / następny** zgodną z kolejnością w tym README. Zmieniając treść jednego doca, zaktualizuj **oba** drzewa.
