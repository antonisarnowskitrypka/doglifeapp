# Ewidencja sprzętu

Narzędzie dla usługodawców do prowadzenia ewidencji sprzętu, historii serwisowania i przypomnień o przeglądach. Szczególnie przydatne dla fizjoterapeutów (bieżnia wodna, Indiba) i trenerów sportowych (sprzęt agility).

## Dodawanie sprzętu

Usługodawca może dodawać pozycje sprzętu do swojej organizacji. Każda pozycja może zawierać:

- Nazwę i opis
- Dane zakupu / nabycia
- Historię serwisowania
- Przypomnienia o przeglądzie / konserwacji

## Widoczność publiczna

Każda pozycja sprzętu ma przełącznik widoczności („oczko"). Po włączeniu pozycja jest pokazywana publicznie na profilu usługodawcy — przydatne do reklamowania możliwości (np. „mamy bieżnię wodną").

Gdy ukryta, pozycja służy wyłącznie do wewnętrznej ewidencji.

Publiczny sprzęt **wzmacnia też wyszukiwanie** — jego nazwy trafiają do tagów firmy, dzięki czemu klienci szukający np. „bieżnia wodna" łatwiej Cię znajdą. Ukryty sprzęt nie wpływa na wyszukiwanie.

## Serwis i konserwacja

Dla każdej pozycji usługodawca może zapisywać zdarzenia serwisowe i ustawiać przypomnienia o nadchodzących przeglądach lub konserwacji.

- Przypomnienia są dostarczane **in-app**.
- Każdy rekord (sama pozycja oraz każde zdarzenie serwisowe) obsługuje **załączniki plików** — faktury, protokoły serwisowe, certyfikaty przeglądów, zdjęcia.

## Przypomnienia

Gdy zbliża się termin przeglądu/konserwacji, usługodawca otrzymuje powiadomienie in-app (zob. [Powiadomienia](./16-notifications.md)).
